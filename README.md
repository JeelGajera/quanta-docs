# QuantaJS Documentation Site

![QuantaJS Docs Banner](./public/img/quantajs_banner.png)

Welcome to the official documentation site for [QuantaJS](https://github.com/quanta-js/quanta), a compact, scalable, and developer-friendly state management library for JavaScript. This documentation site, built with Next.js and MDX, provides a sleek, responsive, and feature-rich experience to explore QuantaJS reactivity, API, integrations, and production guides for the stable `2.x` release line.

## 📦 Getting Started

### Prerequisites

- Node.js (v20 or higher recommended for Next.js 16)
- npm (v9 or higher)

### Installation

1. **Clone the Repository**:
   ```bash
   git clone https://github.com/quanta-js/quanta-docs.git
   cd quanta-docs
   ```
2. **Install Dependencies**:
    ```bash
    npm install
    ```

3. **Run Development Server**:
    ```bash
    npm run dev
    ```

    Open http://localhost:3000 to see the docs in action!
